Overview******

This project is a Personal Portfolio Website created to showcase my web development skills, academic history, professional achievements, and personal projects. 
The portfolio is fully responsive, visually appealing, and user-friendly. 
It highlights my abilities as a developer and serves as a platform for sharing my resume and project links.

Features*****

Navigation Tabs: A smooth navigation experience with Home, About Me, and Portfolio sections accessible through a dynamic tabbed interface.
Document Downloads: Visitors can download my resume, academic transcript, and certifications.
Project Showcases: Highlights two of my projects: a Tic Tac Toe game and a Weather App, complete with descriptions and links.
Responsive Design: Fully functional across various devices and screen sizes.
Social Links: Quick access to my LinkedIn profile and email for networking.

Technologies Used********

Frontend
HTML5: Structure and semantic elements.
CSS3: Styling, animations, and responsiveness.
JavaScript: Functional logic for interactivity and hamburger menu.

Key Features Desciption*******

HOME SECTION
Introduction and a brief description of the portfolio purpose.
Links to download documents: Resume, A+ Certificate, Computer Skills Certificate, and Academic Transcript.

ABOUT SECTION
Detailed academic history and skills list.
A personal statement showcasing my learning journey and aspirations.

PORTFOLIO SECTION
Tic Tac Toe Game:
Built using HTML, CSS, and JavaScript.
Interactive two-player game with a restart button and hover effects.

Weather App:
Built using HTML, CSS, JavaScript, and a Weather API.
Displays real-time weather details, including humidity, wind speed, and cloud cover.

Responsive Design********

This website is designed to look great and function well on all devices:

Desktop: Wide layout with navigation tabs and detailed content sections.
Tablet: Adjusted spacing and font sizes for readability.
Mobile: A hamburger menu for navigation and stacked layouts for content.
