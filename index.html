My Projects

Tic Tac Toe

I developed this Tic Tac Toe game using HTML, CSS and implemented its functionality with JavaScript. It is a 2 player game which has a hover effect that shows whose turn it is to play. Once the game is finished either via a win or a draw, a restart butt on pops up, which resets the game.

Weather App

I developed this weather app using HTML, CSS and JavaScript. CSS was used mainly to implement the style of the app, the functionality is implemented with JavaScript. I used a weather API to be able to access the locations that can be found through the search button. The app also shows the details of the weather condition such as the humidity, wind speed and cloud cover.
